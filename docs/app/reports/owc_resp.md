Пересчет давления на глубину с учетом инклинометрии скважины.

### Параметры для выгрузки:
- **Выбор месторождения** - выбор месторождения для расчета;
- **Выбор объекта** - выбор объекта для расчета;
- **Номер скважины** – номер скважины для расчета;
- **Давление (атм)** - давление на глубине для пересчета;
- **Глубина (м)** - глубина замера давления;
- **Выбор вида исследования** - вид исследования для заполнения аналитики;
- **На дату** - дата замера давления.

### Формат выходных файлов:
- **owc_resp_template.xlsx** - калькулятор для пересчета замеренного давления на глубины ВНК и ВДП:
    * **лист "Пересчет"** - основной лист для расчета;
    * **лист "Глубины"** - инклинометрия скважины;
- **analytics_template.xlsx** - формат аналитики для загрузки в РН-КИН:
    * заполняются только столбцы выделенные желтым цветом.
